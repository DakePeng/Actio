# ASR + 声纹 Recognition System — MVP Implementation Plan (Part 1: Foundation)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Build a working Rust service + Python Worker foundation with gRPC communication, database layer, and API scaffolding — everything needed to add the audio pipeline incrementally.

**Architecture:** Rust controller (axum + tonic + sqlx) talks to Python Worker (gRPC server) over a .proto-defined contract. PostgreSQL with pgvector stores business data and speaker embeddings. WebSocket streams for real-time audio, REST for controls.

**Tech Stack:** Rust 1.94 (axum 0.8, tonic 0.12, prost 0.13, sqlx 0.8, tokio 1.x, tracing 0.1), Python 3.14 (grpcio, funasr, modelscope 3D-Speaker), PostgreSQL + pgvector.

**Key Decisions (from CEO/Eng review):**
- Audio: 16kHz/16-bit/mono PCM, 600ms chunks (9600 samples)
- CAM++ embedding: 192-dim, Z-Norm threshold at 0.0
- gRPC: bidirectional streaming for ASR, unary for VAD/speaker
- Circuit breaker: 3-state (Closed/Open/Half-Open), 3 fails → Open for 30s
- Speaker tag delay: transcript emits with [UNKNOWN], backfills ~2s later
- No auth for MVP, no cloud ASR for MVP, no diarization for MVP

**Prerequisites:**
- PostgreSQL running locally with `pgvector` extension installed
- Python 3.14 with venv capability
- Rust 1.94 installed
- `protoc` installed (`sudo apt install protobuf-compiler`)

```protobuf
// proto/inference.proto

syntax = "proto3";
package inference;

// ============================================================
// VAD Service — detects speech in audio stream
// ============================================================
service VADService {
  rpc DetectSpeech(stream AudioChunk) returns (stream VADResult);
}

message AudioChunk {
  bytes audio_data = 1;       // 16kHz/16bit/mono PCM, ~600ms
  int64 timestamp_ms = 2;
  int32 sequence_num = 3;
  string session_id = 4;
}

message VADResult {
  bool is_speech = 1;
  int64 segment_start_ms = 2;
  int64 segment_end_ms = 3;
  float confidence = 4;
  string session_id = 5;
}

// ============================================================
// ASR Service — streaming speech recognition
// ============================================================
service ASRService {
  // Bidirectional streaming: client sends audio, server streams results
  rpc StreamRecognize(stream AudioChunk) returns (stream RecognizeResult);
  // Unary: send entire audio file, get full transcription
  rpc RecognizeFile(RecognizeFileRequest) returns (RecognizeFileResponse);
}

message RecognizeFileRequest {
  bytes audio_data = 1;       // Full audio file (raw PCM or WAV header)
  string language = 2;        // "zh", "en", etc.
  bool streaming = 3;         // If true, use streaming-optimized model
}

message RecognizeFileResponse {
  repeated TranscriptSegment segments = 1;
}

message RecognizeResult {
  string text = 1;
  bool is_final = 2;
  int64 start_ms = 3;
  int64 end_ms = 4;
  string session_id = 5;
  // For partial results that may be corrected
  string correction_key = 6;
}

message TranscriptSegment {
  string text = 1;
  int64 start_ms = 2;
  int64 end_ms = 3;
  bool is_final = 4;
}

// ============================================================
// Speaker Service — speaker embedding and verification
// ============================================================
service SpeakerService {
  rpc ExtractEmbedding(ExtractEmbeddingRequest) returns (EmbeddingResponse);
  rpc VerifySpeaker(VerifySpeakerRequest) returns (VerifySpeakerResponse);
  rpc HealthCheck(HealthCheckRequest) returns (HealthCheckResponse);
}

message ExtractEmbeddingRequest {
  bytes audio_data = 1;       // Audio for embedding extraction
  float sample_rate = 2;      // 16000.0
}

message EmbeddingResponse {
  repeated float embedding = 1;  // 192-dim for CAM++
  float quality_score = 2;
  float duration_ms = 3;
}

message VerifySpeakerRequest {
  bytes audio_data = 1;       // Audio to verify
  repeated float reference_embedding = 2;  // Stored speaker embedding
}

message VerifySpeakerResponse {
  float similarity_score = 1;
  float threshold = 2;
  bool accepted = 3;
}

message HealthCheckRequest {}

message HealthCheckResponse {
  bool ready = 1;
  bool vad_loaded = 2;
  bool asr_loaded = 3;
  bool speaker_loaded = 4;
  string error_detail = 5;
}
```

```python
# python-worker/main.py
import asyncio
import logging
from concurrent import futures

import grpc
from grpc_health.v1 import health, health_pb2, health_pb2_grpc

from config import WorkerConfig
import inference_pb2_grpc

logger = logging.getLogger("actio-worker")


async def serve() -> None:
    config = WorkerConfig.from_env()
    logger.info(f"Starting worker on {config.host}:{config.port}")
    
    server = grpc.aio.server(futures.ThreadPoolExecutor(max_workers=10))
    
    # Health check service
    health_servicer = health.HealthServicer()
    health_pb2_grpc.add_HealthServicer_to_server(health_servicer, server)
    health_servicer.set("", health_pb2.HealthCheckResponse.SERVING)
    
    # Inference services — will be added in later tasks
    # inference_pb2_grpc.add_VADServiceServicer_to_server(VADService(), server)
    # inference_pb2_grpc.add_ASRServiceServicer_to_server(ASRService(), server)
    # inference_pb2_grpc.add_SpeakerServiceServicer_to_server(SpeakerService(), server)
    
    server.add_insecure_port(f"{config.host}:{config.port}")
    await server.start()
    logger.info(f"Worker started on {config.host}:{config.port}")
    
    try:
        await server.wait_for_termination()
    except KeyboardInterrupt:
        logger.info("Shutting down worker...")
        await server.stop(grace=5.0)

if __name__ == "__main__":
    logging.basicConfig(level=logging.INFO)
    asyncio.run(serve())
```

// src/repository/db.rs
use anyhow::Context;
use sqlx::postgres::PgPoolOptions;
use sqlx::PgPool;
use tracing::info;

pub async fn create_pool(database_url: &str) -> anyhow::Result<PgPool> {
    info!(url = database_url, "Connecting to database");
    
    let pool = PgPoolOptions::new()
        .max_connections(10)
        .acquire_timeout(std::time::Duration::from_secs(5))
        .connect(database_url)
        .await
        .context("Failed to connect to database")?;
    
    info!("Database connected");
    Ok(pool)
}

pub async fn run_migrations(pool: &PgPool) -> anyhow::Result<()> {
    info!("Running database migrations");
    
    sqlx::migrate!("./migrations")
        .run(pool)
        .await
        .context("Migration failed")?;
    
    info!("Migrations complete");
    Ok(())
}

#[tokio::test]
async fn test_speaker_crud() {
    let pool = setup_test_db().await;
    let tenant_id = Uuid::new_v4();
    
    let created = speaker::create_speaker(&pool, "Test Speaker", tenant_id).await.unwrap();
    assert_eq!(created.display_name, "Test Speaker");
    
    let fetched = speaker::get_speaker(&pool, created.id).await.unwrap();
    assert_eq!(fetched.id, created.id);
    
    let all = speaker::list_speakers(&pool).await.unwrap();
    assert!(!all.is_empty());
}

#[tokio::test]
async fn test_session_lifecycle() {
    let pool = setup_test_db().await;
    let tenant_id = Uuid::new_v4();
    
    let session = session::create_session(&pool, tenant_id).await.unwrap();
    assert!(session.ended_at.is_none());
    
    session::end_session(&pool, session.id).await.unwrap();
    
    let fetched = session::get_session(&pool, session.id).await.unwrap();
    assert!(fetched.ended_at.is_some());
}

#[tokio::test]
async fn test_transcript_lifecycle() {
    let pool = setup_test_db().await;
    let session = session::create_session(&pool, Uuid::new_v4()).await.unwrap();
    
    let t = transcript::create_transcript(
        &pool, session.id, "Hello world", 0, 2000, false, None
    ).await.unwrap();
    assert!(!t.is_final);
    assert_eq!(t.text, "Hello world");
    
    let finalized = transcript::finalize_transcript(&pool, t.id, "Hello world.").await.unwrap();
    assert!(finalized.is_final);
    assert_eq!(finalized.text, "Hello world.");
